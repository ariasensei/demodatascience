What is Data Science?

Definition

Data Science is the field that uses data to find insights, make predictions, and support decisions. It works with organized data like spreadsheets (structured) and messy data like social media posts (unstructured) using scientific methods and computer tools.

Why It Matters

Every industry relies on data to:

Main Ideas in Data Science

üìä

Data

Raw facts and information that can be collected, stored, and analyzed.

Examples: Temperature readings, sales numbers, customer feedback, survey responses

üîß

Data Processing

Cleaning, organizing, and preparing data for analysis.

Includes: Removing errors, handling missing values, making data look the same

üîç

Analysis

Examining data to identify trends, patterns, and relationships.

Methods: Statistical analysis, correlation studies, hypothesis testing

üìà

Visualization

Creating visual representations of data through charts, graphs, and dashboards.

Tools: Bar charts, line graphs, pie charts, heat maps, scatter plots

ü§ñ

Machine Learning

Teaching computers to learn from data and make predictions or decisions.

Applications: Recommendation systems, image recognition, fraud detection

üí°

Insights

Actionable knowledge derived from data analysis that helps guide better choices.

Outcomes: Business strategies, policy changes, product improvements

from collections import Counter
fruits = [row["Favorite Fruit"] for row in data]
fruit_counts = Counter(fruits)
print("Fruit Popularity:")
for fruit, count in fruit_counts.items():
    print(f"{fruit}: {count}")
# Step 3: Visualize

plt.ylabel("Number of Votes")
